## 最终接线确认

| 语音模块引脚 | 树莓派GPIO | 物理Pin | 功能 | 触发方式 |
|-----------|-----------|--------|------|---------|
| PA_1 | GPIO17 | Pin 11 | 前进 | 上升沿 |
| PA_4 | GPIO22 | Pin 15 | 停止 | 上升沿 |
| PC_4 | GPIO23 | Pin 16 | 后退 | 上升沿 |
| PB_5 | GPIO24 | Pin 18 | 左转 | 上升沿 |
| PB_6 | GPIO25 | Pin 22 | 右转 | 上升沿 |
| PA_2 | GPIO27 | Pin 13 | 摄像头开关 | 上升沿开 / 下降沿关 |

**PA_2 电平逻辑**：
- 默认状态：低电平（`PUD_DOWN`下拉确保）
- 说"打开摄像头"时：低→高（上升沿）→ 执行开摄像头
- 说"关闭摄像头"时：高→低（下降沿）→ 执行关摄像头

## 带避障的部分

| 模块 | 引脚 | 树莓派GPIO | 物理Pin | 说明 |
|-----|------|-----------|--------|------|
| **超声波开关** | PC6 | GPIO26 | Pin 37 | 高电平开，低电平关 |
| **HC-SR04** | Trig | GPIO16 | Pin 36 | 触发信号输出 |
| **HC-SR04** | Echo | GPIO12 | Pin 32 | 回波信号输入 |
| **HC-SR04** | VCC | 5V | Pin 2/4 | 供电 |
| **HC-SR04** | GND | GND | Pin 6/9/14/20/25/30/34/39 | 接地 |

> **注意**：HC-SR04的Echo输出5V，树莓派GPIO只耐受3.3V。建议加一个**电平转换器**或**电阻分压**（2kΩ+1kΩ），否则可能损坏树莓派GPIO12。

## 避障逻辑说明

```
用户说"前进"
    → 电机前进
    → 超声波启用（PC6高电平）
    → 后台线程每0.1秒测距
    
    距离 < 10cm?
    ├─ 是 → 停止 → 后退0.5秒 → 随机左转/右转0.8秒 → 恢复前进
    └─ 否 → 继续前进
    
用户说"停止"
    → 电机停止
    → 超声波禁用（PC6低电平）
```

代码已完整集成，保存为 `voice_car_ultrasonic.py` 后直接运行即可。
